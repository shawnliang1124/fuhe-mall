# fuhe-mall  

## 简介
fuhe-mall商城，项目技术架构使用springboot 2.1.0 + MybatisPlus+Spring Cloud全家桶等技术，目标是帮助初中级开发工程师熟练掌握框架的基本使用，并且掌握部分的框架原理
